#Task 1

Using pure JS (ES6 is allowed) to create and append to DOM the following elements.

• text input (value - “data”)
• number input (“seconds”)
• button (any text)

GIVEN the text input and number input are filled
WHEN the user clicks on button
THEN the user will see alert message with “data” in not early than “seconds” seconds.
Implement this task using three approaches:

a) Callback
b) Promise
c) Async / await

Hint – create separate function for handling button click, which will extract data from input and handling further logic.

#Task 2, 3

Create a function ‘httpGet’, which imitates an http request. Signature:
httpGet(url, data), where:
url – is some url to use.
data – is data which should be returned.
This function should be promisified (so that it can be used with .then methods or async /
await).
This function should randomly either resolve the passed data or throw error / reject
promise.
The execution time of this function should also be random, 1-3 seconds
Create an application employee, which will initially render a button.
GIVEN the user presses the button
THEN the user will see a spinner
THEN the data will be fetched (using the httpGet function from task II)
IF data fetching is successful – hide spinner and display employees list
IF data fetching is unsuccessful – hide spinner, display error message and allow the user to
try again
HINT: save fake data in a separate file, import it in your js application and pass it as a second
argument to your httpGet function.
